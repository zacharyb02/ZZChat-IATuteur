INTRODUCTION TO CONVOLUTIONAL NEURAL NETWORKS (CNN)
====================================================

Q1: What is a CNN (Convolutional Neural Network)?
A: A CNN is a type of deep neural network specifically designed to process grid-structured data, such as images. It uses convolution operations to automatically extract hierarchical features from input data.

Q2: Why are CNNs effective for image processing?
A: CNNs are effective because they exploit three key properties:
- Weight sharing: reduces the number of parameters
- Local connectivity: each neuron connects to only a local region
- Translation invariance: detects the same patterns regardless of position

Q3: What are the main layers of a CNN?
A: Main layers include:
1. Convolutional Layer (Conv Layer)
2. Pooling Layer (Max/Avg Pooling)
3. Activation Layer (ReLU, Sigmoid, etc.)
4. Fully Connected Layer
5. Normalization Layer (Batch Norm)

Q4: How to create a simple CNN in PyTorch?
A: Code example:

import torch
import torch.nn as nn

class SimpleCNN(nn.Module):
    def __init__(self, num_classes=10):
        super(SimpleCNN, self).__init__()
        
        self.conv1 = nn.Conv2d(3, 32, kernel_size=3, padding=1)
        self.relu1 = nn.ReLU()
        self.pool1 = nn.MaxPool2d(2, 2)
        
        self.conv2 = nn.Conv2d(32, 64, kernel_size=3, padding=1)
        self.relu2 = nn.ReLU()
        self.pool2 = nn.MaxPool2d(2, 2)
        
        self.fc1 = nn.Linear(64 * 8 * 8, 128)
        self.relu3 = nn.ReLU()
        self.fc2 = nn.Linear(128, num_classes)
    
    def forward(self, x):
        x = self.pool1(self.relu1(self.conv1(x)))
        x = self.pool2(self.relu2(self.conv2(x)))
        x = x.view(x.size(0), -1)
        x = self.relu3(self.fc1(x))
        x = self.fc2(x)
        return x

Q5: How to calculate the output size of a convolutional layer?
A: Formula: Output_size = ((Input_size - Kernel_size + 2*Padding) / Stride) + 1

Example:
import torch
import torch.nn as nn

conv = nn.Conv2d(3, 16, kernel_size=5, padding=2, stride=1)
x = torch.randn(1, 3, 32, 32)
output = conv(x)
print(f"Output shape: {output.shape}")  # [1, 16, 32, 32]

Q6: How to initialize CNN weights?
A: 
import torch.nn.init as init

def init_weights(m):
    if isinstance(m, nn.Conv2d):
        init.kaiming_normal_(m.weight, mode='fan_out', nonlinearity='relu')
        if m.bias is not None:
            init.constant_(m.bias, 0)
    elif isinstance(m, nn.Linear):
        init.normal_(m.weight, 0, 0.01)
        init.constant_(m.bias, 0)

model = SimpleCNN()
model.apply(init_weights)

Q7: What is the difference between CNN and MLP?
A: 
- MLP: Fully connected, very high number of parameters for images
- CNN: Local connections with weight sharing, much fewer parameters
- CNN: Exploits spatial structure of images
- MLP: Treats image as 1D vector, loses spatial information

Q8: What is a receptive field?
A: The receptive field is the region of the input image that influences the activation of a given neuron. It increases with network depth.
