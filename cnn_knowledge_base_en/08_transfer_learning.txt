TRANSFER LEARNING
==================

Q1: What is Transfer Learning?
A: Technique using a pretrained model on large database (e.g. ImageNet) as starting point for a new task. Very effective with limited data.

Q2: When to use Transfer Learning?
A:
- Small dataset (< 10K images): ESSENTIAL
- Medium dataset (10K-100K): RECOMMENDED  
- Large dataset (> 100K): OPTIONAL but speeds up training
- Domain similar to ImageNet: VERY EFFECTIVE

Q4: How to use different learning rates?
A:
import torch.optim as optim

model = models.resnet18(pretrained=True)
model.fc = nn.Linear(model.fc.in_features, 10)

# Different learning rates for each part
params = [
    {'params': model.layer1.parameters(), 'lr': 1e-5},
    {'params': model.layer2.parameters(), 'lr': 1e-5},
    {'params': model.layer3.parameters(), 'lr': 1e-4},
    {'params': model.layer4.parameters(), 'lr': 1e-4},
    {'params': model.fc.parameters(), 'lr': 1e-3}
]

optimizer = optim.Adam(params)
